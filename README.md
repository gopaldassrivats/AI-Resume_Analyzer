# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, Gemini API, and RAG concepts.

## Features

- Upload Resume (PDF)
- Enter Job Description
- ATS Score Analysis
- Missing Skills Detection
- Resume Improvement Suggestions

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- LangChain
- FAISS

## Project Structure

- app.py - Streamlit UI
- resume_parser.py - Extract text from resumes
- llm.py - Gemini integration
- rag.py - RAG pipeline
- requirements.txt - Dependencies

## Installation

```bash
pip install -r requirements.txt
